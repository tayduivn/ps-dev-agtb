Overview

Creates a record list from a saved report.

Query String Parameters

This endpoint does not accept any query string parameters.

Input Parameters

Name Type Description Required
<record ID> <string> Target record ID. True.

Input Example




Result



    
    

        Name
        Type
        Description
    

    
    
    

    

        
            record
        
        
            Array
        
        
            The record linked to related records.
        
    

    

        
            records
        
        
            Array
        
        
            Record ids in the record list.
