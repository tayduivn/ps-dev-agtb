Chosen  Kickstrap add-on from Harvest [ Docs]

About

Chosen is a Javascript library from Harvest which changes the look and operation of <select> elements.

Download file

Examples




Code

To enable Chosen, put the following before the ending body tag of the html document:

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
<script type="text/javascript"> $(".chzn-select").chosen(); $(".chzn-select-deselect").chosen({allow_single_deselect:true}); </script>

Note: The first line is a link to jQuery and is only necessary if not included in the existing doc. The jQuery link must be referenced before the Chosen links.
